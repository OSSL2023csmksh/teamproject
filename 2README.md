# mini project 주제
  야구선수들의 능력치를 정리하는 프로그램
# mini project 에 대한 소개
  ##선수별 능력치(타율,타점,홈런)을 입력 받고 알아보기 쉽게 코드를 만들어서 선수별 스탯 확인도 가능하게 하고 스탯 별로 선수의 순위도 보여준다.
  ##추가적으로 모든 선수들의 경기결과에 대한 영향력을 알아내서 예상 mvp 순위를 보여준다.
